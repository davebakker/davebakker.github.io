RETRO GAME

Retro game is built on the idea of completing and releasing a polished product. I achieved to create something fun and awesome while learning about optimization, quick-prototyping, and clean coding. The concept originates from an interest towards the '80s and arcade cabinets. The game is built in Unity, and all the assets are created in Pyxel Edit.
